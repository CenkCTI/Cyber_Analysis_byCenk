# RSS Data Collection and AI Analysis System

This project I developed is an RSS data collection tool (RSS feed parser), initially intended to run on a secure system like Qubes OS. Its purpose is to gather news and content data from specified web links and store them in SQLite (.db) format.

The collected data includes:

Title
Content
Published Date
Source

After storing the data, the project uses an Artificial Intelligence (AI) module to analyze it. Through this analysis:

Connections between past and present data are identified
Potential future trends and relationships are predicted
Details that human analysts might overlook are detected

As a result, the tool helps analysts produce more accurate and comprehensive intelligence.
